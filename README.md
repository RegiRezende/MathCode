# Numerical Methods and Computational Modeling

This repository contains codes, spreadsheets, and simulation examples used in courses on **Numerical Methods** and **Computational Modeling**.

The objective is to provide a collection of educational materials covering fundamental numerical algorithms and their application in engineering and scientific computing.

Topics range from basic numerical techniques, such as root finding and linear systems, to more advanced applications including **computational fluid dynamics (CFD)**.

## Software and Tools

Examples may include implementations using:

- Python
- MATLAB
- Excel
- Mathematica
- Maple
- OpenFOAM
- ANSYS Fluent
- ANSYS CFX
- Google Colab

## Topics Covered

- Root finding methods
- Linear systems
- Numerical integration
- Interpolation
- Differential equations
- Computational Fluid Dynamics (CFD)

## Purpose

This repository is intended to serve as a **teaching resource for students studying numerical methods and computational modeling**.
